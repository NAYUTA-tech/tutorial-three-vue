## 各種コマンド

### 初回セットアップ

```sh
npm install
```

### 開発用ビルド

```sh
npm run dev
```

* http://localhost:3000/ でアクセスできる
* 実行中は、ソースコード保存時に自動でビルドされる

### 本番用ビルド

```sh
npm run build
```
